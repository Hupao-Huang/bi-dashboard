## 二、各部门详细

### 💰 财务部（最大赢家：合思费控全面爆发）

- **费控管理模块**：恢复上线，并补一串易用性——立即同步、看同步日志、当前进度列、当前审批显示真名、按审批人搜索、单据模板筛选、上次同步时间、默认隐藏已结束单。
- **单据详情大升级**：显示发起人 / 公司 / 部门名字；费用明细看到合思完整原始字段；空字段 / 空页签自适应隐藏；**接入用友总账，直接看到借贷分录**（不用再切系统查凭证）；凭证状态加悬停解释（给非财务同事扫盲）。
- **主体校验**（法人实体填错自动标红）：从只查日常报销单，扩展到付款单 / 费用核销 / 银行支付等 4 类真实花钱单据；配套修钉钉花名册翻页 bug（在职员工 147→394，可校验覆盖率 38.8%→93.5%）。
- **AI 审批建议**（樊雪娇日常报销单）：规则从 6 条做到 13 条（发票抬头 / 税号 / 金额 / 开票时间、差旅超标、补贴上限等），并优化口径减少误判（备用金冲抵、花名册未匹配不冤枉）。
- **口径对齐合思后台**："款付票未到" 1742→219 笔；"借款待还款" 数字与合思后台 1:1。
- **合思机器人**：个人中心待审批只读 → 规则编辑器 → 手动审批 + AI 三色建议（同意 / 转人工 / 驳回）；修了因昵称 vs 真名导致 91% 用户看不到自己单子的大坑（改工号精确匹配）。
- **数据新鲜度**：缓存改成与同步同频（最多差 15 分钟）、全量同步提速约 4 倍、费控页加缓存（2.5 秒 → 12 毫秒）。

### 🖥️ 信息部 / IT / 管理员

- **影刀 RPA 一键同步**：不用再跑影刀控制台，BI 监控页点一下就同步、批量补历史缺失日期、后台排队跑（关浏览器也继续）、同步完自动入库 + 钉钉通知。
- **RPA 监控**：彻底修"明明导了却显示未导入"的 T+1 误报；新增"RPA 处理历史"审计；"已导入"判定支持多表；监控页打开秒开。
- **新增「在线用户」页**：随时看现在谁在用 BI（姓名 / 部门 / IP / 设备 / 在线时长）。
- **需求池上线**：业务提的需求统一收口，列表 / 看板 / 甘特三视图管理，状态变更自动钉钉通知。
- **用户 / 反馈管理**：菜单待审批红色徽标 + KPI 卡可点直达筛选 + 账号审批通过自动钉钉通知本人。
- **运维监控**：新增"看实时日志"按钮、补上销售单每天自动同步的定时任务、修同步任务"撞锁跳过被误判失败"的假告警。
- 新增**销售单核对页**（逐日跟吉客云后台对差异）。

### 🤖 管理层 / 全公司

- **AI 智能助手正式上线**：用大白话问数（"上月电商部卖多少""本周比上周怎么样""哪些货缺货""天猫数据到几号了"），8 类高频问题全覆盖，重复问题 50 毫秒秒回、半夜预热高频题，先给 5 位管理层试用。
- **全站质量 / 提速加固**：测试覆盖 13%→70%；慢查询 30 秒自动超时，防止一条慢 SQL 拖死全站；错误处理补全 45 处；6 个 P0 体检修复；56 个页面打开速度体检；全站视觉风格统一。

### 📈 线下部

- **销量预测体系**：9 大区按 SKU 按月预测，系统自动挑当月最准的算法，并考虑春节 / 国庆等节假日因子，**回测误差从 -15.8% 降到 -4.2%**；鼠标悬停看 13 个月趋势小图、预测偏离自动标红、一键导出带样式 Excel——帮线下提前排产备货。

### 🛒 电商部

- **调拨渠道口径大合并**：清心湖自营 + 猫超寄售这两块按调拨算的业务，以前各看板没算进销售口径，综合看板 / 电商部页 / 货品看板 / 产品利润页**全部补齐**（电商趋势图原来一个月差了 600 多万的缺口补上了）。
- 组合装按真实配比拆成单品计算、视频号渠道正确识别、销售单 21 个自定义字段补全。

### 🏭 供应链 / 采购

- **原料行情期货新模块**：16 个原料 / 包材 / 大宗商品的专业 K 线行情，盯豆粕、豆油、白糖等核心原料价格走势判断采购时机。
- 计划看板新增**单仓缺货率** KPI（以前全仓汇总把单仓缺货藏起来了，西安仓 30 个 SKU、云仓 16 个 SKU 其实都缺货）。
- 快递仓储分析提速（切月份查询 7 秒 → 0.05 秒）。

### 🔧 惠及 6 大部门的通用修正

- **货品看板 KPI**：原来只用销量前 15 名当"全部"算，偏小约 18%、商品种类永远显示 15 种；6 个部门的货品看板同步修正，对齐综合看板。
- **产品利润页**：总额修正（不再把 TOP15 合计当全部）+ 各部门调拨口径对齐。

### ▫️ 其他（各 1–2 项）

- **客服部**：满意率改"整体加权口径"，修了评价量小的店满意率虚低（约 80% → 真实 99%，跟各平台后台对齐）。
- **品牌市场部**：心智渗透看板骨架（预览版，等 RPA 补数据 + 业务给 SKU 清单后真上线）。
- **即时零售部**：从电商部独立成一摊、合并朴朴调拨业务进销售口径。
- **分销部**：经销商客户分析模块全套（高价值客户排名、48 个重点业务名单、客户月度走势下钻、按 SKU 拆组合装）。
- **全员通用**：个人信息页显示"昵称-真名"（虎跑-黄承欢）、钉钉真名全员同步。
