新增采购订单

导入 Excel 模板 → 自动翻译编码、算价 → 确认后在用友建采购订单（开立态，不自动审核）。建单不可撤回，请先看清预览。

1 输入口令解锁

口令向管理员索取。解锁后才能使用，关闭工具后需重新输入。

2 选择填好的 Excel，预览

订单张数 0
可建 0
有问题 0
含税总额 0
行采购组织供应商物料编码物料名称 数量含税单价含税金额税率到货日期状态
红色行有问题不会建单（多为：供应商名单里没有、物料/组织编码查不到、日期或金额填错）。橙色是提醒（已按货品档案税率算）。

3 确认建单（写入用友，不可撤回）

采购组织供应商单据日期明细数结果用友单号
显示“已跳过/10分钟内已提交”是防重复保护。确需重建，点下面按钮强制重发。
